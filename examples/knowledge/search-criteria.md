# Search Criteria Example

Use this file to describe which roles should be considered relevant.

## Role

Primary target: Senior Product Manager.

Acceptable variations:

- Product Manager
- Senior Product Manager
- Lead Product Manager
- AI Product Manager
- FinTech Product Manager

## Seniority

Target seniority:

- Senior
- Lead
- Staff

Avoid:

- Internship roles
- Junior roles
- Roles that are mostly project management rather than product management

## Location

Preferred:

- Remote Europe
- Remote EMEA
- Global remote
- Portugal
- Lisbon hybrid

Exclude:

- US-only remote
- Canada-only remote
- Relocation-required roles

## Preferred Industries

Priority:

1. FinTech
2. AI / AI agents
3. Crypto / Web3
4. Infrastructure / B2B SaaS

## Exclusions

- Exclude roles with no clear remote policy.
- Exclude hybrid roles unless they are in my city.
- Exclude roles focused only on people management with no product ownership.

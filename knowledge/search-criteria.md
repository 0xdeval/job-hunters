## General criteria

### Role

The primarily role to look for is product manager

There could be different variation of the role:

- AI product manager
- Crypto product manager
- Product manager
- etc

But the core role is product manager, noting else

### Seniority

Take into account only the following seniority levels that you can extract from title or other job description parts:

- Product manager
- Senior product manager
- Lead product manager
- Staff product manager
- Associate product manager (only in case if it's enterprise company)

Target to Senior product manager

### Location

Primary location: remote globally, not location specific

Secondary location: Portugal, Lisbon OR/AND EMEA/Europe remote
You can also take into account "Spain - Remote" option for my role

### Preferred companies industries

The top list by priority:

1. Finance/FinTech
2. AI/Agentic
3. Crypto/Web3
4. Infrastructure/B2B SaaS

The mix of different industries works as well

## Critical exclude criteria

1. Job that are remote only in a particular country except of Portugal. For example, exclude if "remote US", "Canada remote only", etc
2. If there is no obvious tag regrading a location, but there is a salary info available, filter job description by a salary range. If it's over $120k+ annually, skip it as it's potentially a US/CA only job
3. If there is a mention of hybrid format of work and the location is not in Portugal, you can skip as I'm not suited for this JD

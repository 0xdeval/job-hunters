# Personal Projects

## Puppeteer - AI avatars automation framework

**Worked period:** March, 2026 - now

### Description

Built a multi-agent social media automation framework where AI-driven personas autonomously fetch news, generate context-aware posts, and reply on X and Facebook — powered by n8n orchestration, OpenRouter LLM routing, and a custom browser-use automation service. the overall system is based on the modular avatar identity system (personality, behavior, tone) injected dynamically into LLM via n8n data pipelines - enabling scallable self-service persona management without code changes

### Tech Stack

- AI/automations stack: n8n, OpenRouter, Claude REST API, OpenAI REST API
- Programming: TypeScript, Playwright, Rest API, Dolphin
